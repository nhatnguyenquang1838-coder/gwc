# Verification Before Completion — GWC Offline Adaptation

Before claiming G3 PASS, verify fresh evidence for the exact current head SHA: validator output, CI conclusion, review verdict, acceptance criteria, finding closure, and delivery record validation.

Any new commit invalidates prior validation, CI, review, and closure evidence. Re-run all required checks before presenting G4.
