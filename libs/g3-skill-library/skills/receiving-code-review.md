# Receiving Code Review — GWC Offline Adaptation

Verify each review finding against the actual diff, requirements, tests, and governance. Accept, reject, or narrow findings with evidence.

Accepted implementation changes return to G2. BLOCKER prevents G3 PASS. MAJOR must be fixed or covered by exact-head human risk acceptance. MINOR may be fixed or deferred with traceability. NIT is recorded as non-blocking.
