# Core Runbooks

Operational runbooks translate canonical governance contracts into step-by-step agent procedures.

## Canonical runbooks

- `GATE_G0_G1_OPERATIONAL_RUNBOOK_v1.0.md` — global procedure for G0_CONTEXT and G1_ALIGNMENT across all agents, projects, and execution modes.

Project and agent instructions may tighten these procedures but must not weaken canonical policy, gate contracts, approval boundaries, protected-branch rules, or HITL requirements.
